# DealRadar Bot

A Telegram bot that aggregates, filters, and scores deals from various sources like Reddit and RSS feeds, and posts them with affiliate links.

## Project Structure
- `app/`: Main application code.
- `data/`: SQLite database storage.
- `main.py`: Entry point.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file from `.env.example` and fill in your `BOT_TOKEN`.
4. Run the bot: `python app/main.py`.
